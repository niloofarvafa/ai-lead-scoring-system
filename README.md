# 🤖 AI Lead Scoring System

This project simulates an AI-powered B2B lead scoring system.

## 🚀 What It Does
- Loads raw B2B leads
- Applies business-driven scoring logic
- Automatically segments leads into High / Medium / Low priority
- Outputs prioritised lead list

## 🧠 Scoring Logic
The scoring model evaluates:
- Company size
- Budget level
- Engagement score

Each factor contributes to a weighted final lead score.

## 📊 Business Value
This system helps sales teams:
- Focus on high-value prospects
- Improve conversion rates
- Reduce wasted outreach efforts

## 🛠 Tech Stack
Python · Pandas · Business Scoring Logic
